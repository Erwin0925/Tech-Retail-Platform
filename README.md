# Tech Retail Platform (MERN Stack)

This project is a simple tech retail platform showcasing products like Apple Watch, iPhones, and AirPods. Users can add, edit, and delete products as they manage their store. The application also supports light and dark modes for a better user experience.

## Live Demo
[Visit the Tech Retail Platform](https://tech-retail-platform.onrender.com/)

## Features
- View current products 
- Add new products 
- Edit or delete existing products
- Light and Dark mode support

## Installation
1. Clone the repository
```
git clone https://github.com/Erwin0925/Tech-Retail-Platform.git
```
3. Navigate to the project directory.
```
cd your-repo-name
 ```
3. Install dependencies
```
npm run build
```
5. Run the Application
```
npm run start
```
